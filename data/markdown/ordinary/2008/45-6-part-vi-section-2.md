ГЎГіВЈвҖ CГЁOВЎ ГқЕ’ГөВЈГ·Лң ГұЕёГҒвҖ№ ГұВЈГөвҖ Г¬ ГқвҖ CЛҶГ®В¬ГҙГөЛң, MВҝЕ В№ГівҖ№
MВҝЕ В№ГівҖ№ ГұВЈГөвҖ Г¬вҖ№, ГӘЖ’ГЁГіВЈВ№ГівҖ№ ГЎГіВЈвҖ C ГҘВЎPГІвҖ№, ГҚГіВЈЛңГ°ВЈВ¬Г·ГІвҖ№ AГіВЈГұ ГЎГіВЈвҖ C ГұВЎГёЛҶ Г®В¬ГҙГөЛң
MВҝЕ В№ГівҖ№ ГұВЈГөвҖ Г¬вҖ№, ГӘЖ’ГЁГіВЈВ№ГівҖ№ ГЎГіВЈвҖ C ГҘВЎPГІвҖ№, ГҚГіВЈЛңГ°ВЈВ¬Г·ГІвҖ№ AГіВЈГұ ГЎГіВЈвҖ C ГұВЎГёЛҶ Г®В¬ГҙГөЛң
MВҝЕ В№ГівҖ№ ГұВЈГөвҖ Г¬вҖ№, ГӘЖ’ГЁГіВЈВ№ГівҖ№ ГЎГіВЈвҖ C ГҘВЎPГІвҖ№, ГҚГіВЈЛңГ°ВЈВ¬Г·ГІвҖ№ AГіВЈГұ ГЎГіВЈвҖ C ГұВЎГёЛҶ Г®В¬ГҙГөЛң
MВҝЕ В№ГівҖ№ ГұВЈГөвҖ Г¬вҖ№, ГӘЖ’ГЁГіВЈВ№ГівҖ№ ГЎГіВЈвҖ C ГҘВЎPГІвҖ№, ГҚГіВЈЛңГ°ВЈВ¬Г·ГІвҖ№ AГіВЈГұ ГЎГіВЈвҖ C ГұВЎГёЛҶ Г®В¬ГҙГөЛң
MВҝЕ В№ГівҖ№ ГұВЈГөвҖ Г¬вҖ№, ГӘЖ’ГЁГіВЈВ№ГівҖ№ ГЎГіВЈвҖ C ГҘВЎPГІвҖ№, ГҚГіВЈЛңГ°ВЈВ¬Г·ГІвҖ№ AГіВЈГұ ГЎГіВЈвҖ C ГұВЎГёЛҶ Г®В¬ГҙГөЛң
Г°Г®M cвҖҡГЁвҖ№ ВӘГӘЕ’Г®в„ў Г°ЕёPГІ ГңPMвҖҡВ¬ГЁ.
Г°Г®M cвҖҡГЁвҖ№ ВӘГӘЕ’Г®в„ў Г°ЕёPГІ ГңPMвҖҡВ¬ГЁ.
Г°Г®M cвҖҡГЁвҖ№ ВӘГӘЕ’Г®в„ў Г°ЕёPГІ ГңPMвҖҡВ¬ГЁ.
Г°Г®M cвҖҡГЁвҖ№ ВӘГӘЕ’Г®в„ў Г°ЕёPГІ ГңPMвҖҡВ¬ГЁ.
Г°Г®M cвҖҡГЁвҖ№ ВӘГӘЕ’Г®в„ў Г°ЕёPГІ ГңPMвҖҡВ¬ГЁ.

Г®IвҖәГҜВЈВҙ ГЎГіВЈвҖ CГЁЕ“ ГӘвҖ Г¬вҖ№, 1994 HKВҫ 205, Г вҖ HKВҫ 11-в„ў ГЎГіВЈвҖ CГЁOВЎ ГқЕ’ГөВЈГ·Лң ГұЕёГҒвҖ№ ГұВЈГөвҖ Г¬ ГқвҖ CЛҶГ®В¬ГҙГөВјвҖҡВ°
ГңOвҖҡГЁЕ Г°вҖ ВҙЕ“Г· ГңFГЁВЈГіЛҶFВЎГ°В®, В«Г®CГІ ГЎГіГЁ В«ГөВ¬Гҙ Г ГҒF FвҖ Г¬ЛҶFв„ў, В«ГөВ¬ГҙВ«ГІ ВӘГӘЕ’ГІВЈГұв„ў В«Г°ВЈLГІВЈГ№ ГқГөГӯЖ’ГЁВ¬Г·
Г®ГІВЈKЛҶВ¶, В«ГөВ¬Гҙ ВӘГӘЕ’Г®В¶ В«Г°ВЈВЎГҒ, FГ№ГӘK ГөВјВ¬ГЁ Г°FВ«ГөвҖ В®в„ў, В«ГөВ¬ГҙГІВЈвҖ ГЁOВЎ В¬ГЁВӘГІВҝЛҶВ¬Г®Е  В«Г°ВЈLГІВЈГЁ ГһвҖ ВҙЕ“Г·В¬Г®
ГңГӮГұFЛҶВ¶Е“Г· ГЁВЈГіГӯЛҶFГ№ВЈВҪвҖ№, ГЎГіВЈвҖ C ГұВЎГёЛҶ Г®В¬ГҙГөЛң ГўВЎГӮвҖ№ ГңFГЁВЈГіЛҶВ¬Г®ЛҶ Г®ГөГёВЈГЁЕ  Г°ГІВЎГ°ВҙЛҶF ГңГіВІ FвҖ Г¬ЛҶFв„ў
ВәВ¬ГёВ«ГЁВҙ ВӘГӘЕ’В¶, ГңГіВІ GFВ¬ГІ В¬ГЁГІВЈГ¬в„ў ВӘГӘЕ’В¶, ГһГ®ВЎ ГҚГҙвҖ№ ГЈВ¬Г¶, ГўOГІ AГіВЈГұЕ В№Гё В«ГөВ¬ГҙГІЕёГё ГұвҖҡГЁГҖвҖҡВ° AВ¬Г¬вҖҡГЁВ«ГөвҖЎВ®ГІ
GГІВЈГІГұВЈГ№ В«ГөВ¬Гҙ ГөВЈЕ’В¬Г° Г°PЛҶВ¶, ГңГіCВЎ YKГІ FвҖ Г¬ГұВЈГ№, В«Г®CГІ ГЎГіГЁ В«ГөВ¬Гҙ Г ГҒF FвҖ Г¬ЛҶВ¬Г®вҖһ YЛңВ°В¬ГҙЛҶГ®, ГҚГіВЈЛңГ°ВЈВ¬Г·ГІвҖ№
ГЎГіВЈвҖ C ГұВЎГёЛҶ Г®В¬ГҙГөЛң FВј. Гўv. В«Г»вҖҡ ГңвҖҡГ°Лң ГўВЎГ°ГөВ¬Гі ГЎГіВЈвҖ C ГұВЎГёЛҶ Г®В¬ГҙГөЛң Г°Г®MJLВјвҖ°В¶ cвҖҡГЁвҖ№ ВӘГӘЕ’В¶ ГңPMвҖҡВ¬ГЁ
